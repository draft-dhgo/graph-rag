## 서문: 이 교재 사용법

이 교재는 GraphRAG(그래프 기반 검색 증강 생성)의 기본 개념부터 고급 구현까지 단계적으로 안내하도록 설계되었습니다. 각 파트는 이전 내용을 바탕으로 구축되어 점진적인 학습 경험을 제공합니다.

**전제 조건:**
- Python 프로그래밍 기본 지식
- 머신러닝 개념에 대한 익숙함
- 그래프 이론 사전 지식 불필요

**학습 경로:**
```
Part 1: 기초 → Part 2: 핵심 개념 → Part 3: 구현 → Part 4: 고급 주제 → Part 5: 실무 가이드
```
